This is the main page for the monitoring of the CMS Database infrastructure. It publishes the results of all the existing notification systems:

  • CMS Databases
    It monitors the hardware and service infrastructure of both the online and offline Database system.
  • DB Web Applications
    It monitors backend and frontend for all web applications displaying condition data.
  • Drop Box
    It monitors the drop box infrastructure for populating Condition Database with calibration constants.
  • Frontier Offline
    It monitors the Frontier machines and services in the offline network at CERN.
  • Frontier Online
    It monitors the Frontier machines and services in the online network at the CMS experimental area.
  • O2O
    It monitors the jobs running in the centralized account on the CMS private network which perform the population of the offline condition Database reading the online master condition Database.
